## 📦 打包项目文件

### 在本地电脑操作

1. **打包项目**（如果需要手动部署）
```bash
cd /path/to/your/project
tar -czf personality-assessment.tar.gz \
  --exclude='node_modules' \
  --exclude='.next' \
  --exclude='.git' \
  --exclude='complete-project' \
  --exclude='*.tar.gz' \
  .
```

2. **上传到服务器**
```bash
scp personality-assessment.tar.gz root@your-server-ip:/tmp/
```

3. **在服务器解压**
```bash
ssh root@your-server-ip
mkdir -p /opt/personality-assessment
cd /opt/personality-assessment
tar -xzf /tmp/personality-assessment.tar.gz
```

## 🛠️ 手动部署步骤

### 1. 准备工作（在服务器上）

```bash
# 安装 Docker
curl -fsSL https://get.docker.com | sh

# 安装 Docker Compose
curl -L "https://github.com/docker/compose/releases/latest/download/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
chmod +x /usr/local/bin/docker-compose

# 安装 Nginx
yum install -y nginx  # CentOS
# 或
apt update && apt install -y nginx  # Ubuntu
```

### 2. 上传项目文件

```bash
# 创建项目目录
mkdir -p /opt/personality-assessment
cd /opt/personality-assessment

# 上传项目文件（选择一种方式）
# 方式 A: 使用 Git
git clone <your-repo-url> .

# 方式 B: 使用 SCP (在本地电脑)
# scp -r project/ root@server:/opt/personality-assessment/
```

### 3. 启动服务

```bash
cd /opt/personality-assessment

# 构建并启动
docker-compose build
docker-compose up -d

# 查看状态
docker-compose ps
docker-compose logs -f
```

### 4. 配置 Nginx

创建配置文件 `/etc/nginx/conf.d/personality.conf`:

```nginx
server {
    listen 80;
    server_name yourdomain.com;  # 改成你的域名
    
    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

```bash
# 测试并重启 Nginx
nginx -t
systemctl restart nginx
```

### 5. 配置 SSL（可选但推荐）

```bash
# 安装 Certbot
yum install -y certbot python3-certbot-nginx  # CentOS
apt install -y certbot python3-certbot-nginx   # Ubuntu

# 申请证书
certbot --nginx -d yourdomain.com

# 设置自动续期
systemctl enable certbot.timer
```

## 🌐 域名配置

### 第一步：购买域名
在阿里云、腾讯云等平台购买域名

### 第二步：DNS解析配置

登录域名管理平台，添加A记录：

```
主机记录：@ （或 www，或其他子域名）
记录类型：A
线路类型：默认
记录值：你的服务器公网IP
TTL：600
```

### 第三步：验证解析

```bash
# 在本地电脑测试
ping yourdomain.com

# 应该能解析到服务器IP
```

## 📞 常见问题

### 无法访问网站？

1. **检查防火墙**
```bash
# 开放端口
firewall-cmd --permanent --add-service=http
firewall-cmd --permanent --add-service=https
firewall-cmd --reload
```

2. **检查服务是否运行**
```bash
docker-compose ps
curl http://localhost:3000
```

3. **检查 Nginx 状态**
```bash
systemctl status nginx
nginx -t
```

### 502 Bad Gateway？

```bash
# 重启服务
cd /opt/personality-assessment
docker-compose restart

# 查看日志
docker-compose logs
```

### 如何更新？

```bash
cd /opt/personality-assessment
git pull  # 或重新上传文件
docker-compose build --no-cache
docker-compose up -d
```
