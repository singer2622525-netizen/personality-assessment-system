# 🚀 部署方式说明

## 方式一：一键发布脚本（推荐）⭐

这是**最简单快捷**的部署方式，自动完成构建、打包、上传、部署全流程。

### 使用步骤

1. **打开终端，切换到项目根目录**：
   ```bash
   cd "/Users/a1/Documents/Projects/cursor/招聘人格评测结果智能分析"
   ```

2. **执行一键发布脚本**：
   ```bash
   "/Users/a1/Documents/Projects/cursor/00-03-网页程序部署程序在dell 服务器（通用部署环境）/脚本/服务器部署/deploy-build-and-upload.sh"
   ```

### 脚本自动完成的工作

1. ✅ 本地 `npm install`（安装依赖）
2. ✅ 本地 `npm run build`（构建项目）
3. ✅ 打包构建产物（`.next/standalone`, `.next/static`, `public`, `package.json`）
4. ✅ 上传到服务器 `~/release/build-only.tar.gz`
5. ✅ 在服务器执行部署脚本：
   - 备份当前版本
   - 解包新版本
   - 重启 PM2 服务
   - 健康检查

### 优点

- ✅ **自动化**：一条命令完成所有步骤
- ✅ **快速**：只上传构建产物，体积小
- ✅ **安全**：自动备份，可回滚
- ✅ **验证**：自动健康检查

## 方式二：手动部署包（备选）

如果一键发布脚本不可用，可以使用手动部署包。

### 使用步骤

1. **上传部署包到服务器**：
   ```bash
   scp 完整部署包.tar.gz songtuo@192.168.13.44:~/
   ```

2. **SSH连接到服务器**：
   ```bash
   ssh songtuo@192.168.13.44
   ```

3. **在服务器上执行**（详见 `完整部署包-部署说明.md`）：
   ```bash
   # 备份
   cd ~/personality-assessment
   cp -r . ../personality-assessment.backup.$(date +%Y%m%d_%H%M%S)

   # 解压
   cd ~
   tar -xzf 完整部署包.tar.gz

   # 复制文件
   cp -r 招聘人格评测结果智能分析/app ~/personality-assessment/
   cp -r 招聘人格评测结果智能分析/lib ~/personality-assessment/
   cp 招聘人格评测结果智能分析/*.json ~/personality-assessment/
   cp 招聘人格评测结果智能分析/*.js ~/personality-assessment/
   cp 招聘人格评测结果智能分析/*.ts ~/personality-assessment/

   # 构建
   cd ~/personality-assessment
   npm install
   npm run build

   # 重启
   pm2 restart all
   ```

### 适用场景

- 一键发布脚本不可用
- 需要手动控制部署过程
- 服务器环境特殊，需要自定义部署
