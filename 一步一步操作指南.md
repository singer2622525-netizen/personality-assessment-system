# 🎯 GitHub Desktop 一步一步操作指南

## 📍 当前步骤：添加现有本地仓库

### 方法 1：使用按钮添加（详细步骤）

#### 步骤 1：找到添加按钮
1. **在 GitHub Desktop 主界面**
2. **看右侧区域**，有 4 个大按钮
3. **找到这个按钮**（从上往下数第 4 个）：
   ```
   ┌─────────────────────────────────────┐
   │  📁 Add an Existing Repository     │
   │     from your Local Drive...        │
   └─────────────────────────────────────┘
   ```
4. **点击这个按钮**

#### 步骤 2：选择项目文件夹
点击按钮后，会弹出一个对话框：

1. **在对话框中，你会看到：**
   - "Local Path" 或 "Choose a folder" 的输入框
   - 旁边有一个 **"Choose..."** 或 **"浏览..."** 按钮

2. **点击 "Choose..." 按钮**

3. **在文件选择窗口中：**
   - 按 `Cmd + Shift + G`（这会打开"前往文件夹"）
   - 或者直接在左侧边栏找到 "Documents"
   - 然后依次进入：
     - `Documents`
     - `Projects`
     - `cursor`
     - `招聘人格评测结果智能分析`
   - **选择 "招聘人格评测结果智能分析" 这个文件夹**
   - 点击"打开"或"选择"

4. **回到对话框，点击 "Add Repository" 按钮**

### 方法 2：拖拽添加（更简单）

#### 步骤 1：打开 Finder
1. 点击 Dock 中的 **Finder 图标**（笑脸图标）
2. 或者按 `Cmd + Space`，输入 "Finder"，按回车

#### 步骤 2：找到项目文件夹
在 Finder 中：

1. **按 `Cmd + Shift + G`**（打开"前往文件夹"）
2. **输入或粘贴这个路径：**
   ```
   /Users/a1/Documents/Projects/cursor/招聘人格评测结果智能分析
   ```
3. **按回车**，会直接跳转到项目文件夹

#### 步骤 3：拖拽到 GitHub Desktop
1. **确保 GitHub Desktop 窗口是打开的**
2. **在 Finder 中，点击项目文件夹**（招聘人格评测结果智能分析）
3. **按住鼠标左键，拖拽这个文件夹**
4. **拖到 GitHub Desktop 窗口上**
5. **松开鼠标**

## 🔍 如果找不到按钮怎么办？

### 检查 1：确认你在主界面
- 如果看到 "Clone a Repository" 对话框，**先关闭它**（点击 X）
- 应该看到主界面，标题是 "Let's get started!"

### 检查 2：查看右侧区域
- 右侧应该有 4 个大按钮
- 从上到下分别是：
  1. Create a Tutorial Repository...
  2. Clone a Repository from the Internet...
  3. Create a New Repository on your Local Drive...
  4. **Add an Existing Repository from your Local Drive...** ← 这个！

### 检查 3：如果按钮被遮挡
- 尝试滚动右侧区域
- 或者调整窗口大小

## ❓ 常见问题

### Q: 我找不到 "Add an Existing Repository" 按钮
**A:**
- 确认你在主界面（不是对话框）
- 查看右侧，应该能看到 4 个按钮
- 如果还是找不到，试试拖拽方法

### Q: 我点击了按钮，但是不知道选哪个文件夹
**A:**
- 在文件选择窗口中，按 `Cmd + Shift + G`
- 粘贴路径：`/Users/a1/Documents/Projects/cursor/招聘人格评测结果智能分析`
- 按回车，直接跳转

### Q: 拖拽后没有反应
**A:**
- 确保拖拽的是文件夹本身，不是里面的文件
- 确保 GitHub Desktop 窗口是激活的（点击一下窗口）
- 拖到窗口的任意位置都可以
