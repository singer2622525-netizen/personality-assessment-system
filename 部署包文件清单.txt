🚀 部署步骤
-----------

1. 上传压缩包到服务器
   scp 完整部署包.tar.gz songtuo@192.168.13.44:~/

2. SSH连接到服务器
   ssh songtuo@192.168.13.44

3. 备份现有项目
   cd ~/personality-assessment
   cp -r . ../personality-assessment.backup.$(date +%Y%m%d_%H%M%S)

4. 解压部署包
   cd ~
   tar -xzf 完整部署包.tar.gz

5. 复制文件到项目目录
   # 保留 data/ 目录，只复制代码文件
   cp -r app lib *.json *.js *.ts ~/personality-assessment/

6. 安装依赖（如果需要）
   cd ~/personality-assessment
   npm install

7. 重新构建
   npm run build

8. 重启服务
   pm2 restart all
   # 或
   npm start
