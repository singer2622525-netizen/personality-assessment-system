```typescript

```

#### 问题2：viewport 配置警告 ✅ 已修复
**问题**：Next.js 14 要求 viewport 单独导出，不能放在 metadata 中
**修复**：将 viewport 从 metadata 中分离，单独导出
```typescript
export const viewport: Viewport = {
  width: 'device-width',
  initialScale: 1,
}
```
